How to change the logo?

You can do that very easily, simply replace the current logo image with your own one.

The curren logo image is located in images/logo.png folder. The required size is 140 × 70(px). Note, You have to upload the logo in 2 sizes bigger to look great on tablets and mobile devices.

For example, your site logo is 140x70px the retina version will be 280x140.

How to set up Striped Page?

To change the stripes section, please open index.html file and find line: 19 <figure class="strip-menu" data-width="20" data-count="5">
Under the line you will find all required items. You can easily change the titles and background images, add/edit or delete them from the code. Please check the example below:

               <section class="strip-item" style="background-image:url(img/landing/lan-1-1.jpg)">
                       <div class="strip-fadder"><div>
                       <a href="gallery-kenburns.html" class="wrapped_link"></a>
                       <div class="strip-text">
                              <div>
                                  <h1 class="strip-title">Kenburns</h1>
                                  <h3 class="strip-caption">Fullscreen Image Gallery</h3>
                              </div>
                       </div>
                       <span class="logo"><img src="img/logo.png" alt="" class="logo_def"><img src="img/retina/
                       logo.png" alt="" class="logo_retina"></span>
               </section>

  • style="background-image:url(img/landing/lan-1-1.jpg)"is is the path to the image.
  • <a href="gallery-kenburns.html"is the path to the page.
  • h1 class="strip-title"-the title of the stripe.
  • h3 class="strip-caption"- displays the description text below the title.

How to add/manage Gallery?

As you can see this template comes with different gallery types: Grid, Masonry, Kenburns, Slider and Grid Video.

You can find the required one in templates files (gallery-masonry.html, gallery-grid.html, gallery-kenburns.html, fullscreen-grid-video-gallery.html and slider.html )

Let's try to manage one of the gallery examples. Please open gallery-masonry.html file and find line: 138

                                             <div class="fullscreen_block hided">
                                                <div class="fs_grid_portfolio">
                                                   <div class="grid-port-item">
                                                              <a href="img/gallery/grid/1.jpg" class="
                                                              prettyPhoto" data-rel="prettyPhoto[gallery99]" title="">
                                                                    <img class="gallery-stand-img" src="img/gallery/grid/thumbs/1.jpg"
                                                                    alt="" class=" fw_featured_image" width="540" height="420">
                                                                    <div class="grid-port-fadder"> <div>
                                                                    <div class="grid-port-cont">
                                                                                <span class="grid-port-ico gallery-port-ico"><span>
                                                                    </div>
                                                                 </a>
                                                      </div>

Every gallery item starts and ends with <div></div>

  • <a href="img/gallery/grid/1.jpg"": is the path to the image;
  • <img class="gallery-stand-img" src="img/gallery/grid/thumbs/1.jpg>: is the image source of the thumbnail, its size should be 540x420px
  • alt="Title here": add the image title;
  • Note, change only details indicated above.

How to change the color of the theme?

You can do that easily, in two ways:

  • Auto replacing. In order to change main color theme you should open css/color_theme.css and replace the default color background-color:rgba(109,133,131, 0.85); with your own one.
  • Manually. Simply open the same file and edit each instance of the main color #6d8583

Contact Form Setup

In case you need to use the contacts page with the form where customer can send mail, simply open one of the files: contacts-fullscreen.html, contacts-fullwidth.html, contacts-with-sidebar.html page and find the form:

                                     <form id="ajax-contact-form" action="#" >
                                                    <input type="text "name="name" value="" placeholder="Name *" >
                                                    <input type="text "name="email" value="" placeholder="Email *" >
                                                    <input type="text "name="subject" value="" placeholder="Subject" >
                                                    <textarea name="message" id="message" placeholder="Message *">
                                                   </textarea>
                                                    <input type="submit" value="Send message!">
                                        </form >

Please open mail.php file and find test@test.com on line 3, simply replace to your email address. All the requests will be sent to the adress you indicated. That's all ;)
